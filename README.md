# Dry Flower 電商網站

## 說明
* 電子商務網站實作專案。
* 一頁式網站。
* 包含前後台功能。

## 頁面
### 前台
* 首頁頁面。
* 商品清單頁面。
* 聯繫頁面。
* 各商品詳細資訊頁面。
* 購物車及最愛功能。
* 訂單確認及顧客資訊輸入頁面。
* 付款頁面。

### 後台
* 登入頁面。
* 商品管理頁面。
* 訂單管理頁面。
* 優惠券管理頁面。

## 技術
* Vue.js
* Vue CLI
* Vue Router
* Vuex
* Bootstrap 4
